# Практика по CI для Python веб-приложения
## Цель практики:
- Научиться реализовывать CI пайплайны для GitLab
## Задачи:
- Прописать базовые эндпоинты на фреймворке FastAPI
- Подготовить для них базовые тесты для проверки успешности пайплайнов
- Прописать пайплайны для автоматического тестрирования и валидации:
    - процесса прохождения unit-тестов
    - прохождения читаемости кода по средствам линтеров
    - корректного процесса логирования
## Итог:
- Задачи поставленные для текущей практики были выполненны
- Были освоины базовые концепции создания CI пайплайнов для платформы GitLab
